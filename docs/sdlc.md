# Software Development Life Cycle (SDLC)

We follow an **Agile** methodology for the Smart Task Management System.

## Phases

1.  **Planning**: Define core requirements for the task management system.
2.  **Analysis**: Analyze user needs and define system specifications.
3.  **Design**: Create system architecture, class diagrams, and ER diagrams.
4.  **Implementation**: Develop the application using Next.js and TypeScript.
5.  **Testing**: Perform unit and integration tests.
6.  **Deployment**: Deploy to a cloud platform like Vercel.
7.  **Maintenance**: Regular updates and bug fixes.

## OOP Concepts Used

- **Encapsulation**: Used in classes like `User`, `Project`, and `Task` to keep data and methods together.
- **Inheritance/Interfaces**: Used `ITask` and `IProject` interfaces to define common structures.
- **Polymorphism**: Potential for future use with different task types.
- **Abstraction**: Using interfaces to hide implementation details.
